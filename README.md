# Food_dilvery_system
🍔 Food Delivery System (PHP + MySQL)

A fully functional Food Delivery System built using PHP, MySQL, and XAMPP (phpMyAdmin). This project includes both Admin and User panels with complete end-to-end functionality for managing food ordering and delivery operations.

🚀 Features

👤 User Panel:

User registration & login system
Browse food items and categories
Add items to cart
Place orders
View order history

🛠️ Admin Panel:

Admin login dashboard
Add, update, and delete food items
Manage categories
View and manage customer orders
Track order status
🧰 Technologies Used
PHP (Core PHP)
MySQL Database
XAMPP Server
phpMyAdmin
HTML, CSS, JavaScript
📌 Project Purpose

This project is developed for learning and practicing full-stack web development concepts, especially CRUD operations, authentication, and database management.
